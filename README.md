# Mara Runner V1.0

## Apresento a todos o Mara Runner! Projeto de minha autoria e atualmente em desenvolvimento.

![Captura de tela 2024-08-02 201836](https://github.com/user-attachments/assets/700c014c-52ee-43dc-8f3a-b97bd8a59945)

## Com o objetivo de cumprir o 5º desafio da Trilha de Programaçào de Jogos Digitais do programa Trilhas Inova Maranhão, esse game teste foi construído na Unity Engine utilizando principalmente C#, além de assets gratuitos que podem ser encontrados na internet e nas lojas espepcializadas como a Asset Store. O objetio inicial é aprendizado, mas uma versão com personagens autorais e assets próprios está sendo pensada para o futuro. Portanto essa é atualmente uma versão sem fins comerciais.
